What is this?
---
At its core, this is an extensible experience framework made for developers of Garry's mod game modes. Although relatively simple by design, it's clever architecture will allow any developer to tap into its capabilities with just a few lines of code.

How to use it?
---
Using the this framework in any game mode is quick and easy. Just "include()" the files on the client (cl_expsys.lua) and server (expsys.lua) in your game mode respectively. Afterwards, you are free to use the experience system anywhere in your game mode! Check out the GitHub wiki for info on the available functions and hooks that this framework has to offer.
